# 快捷键

zeroDraw 支持以下键盘快捷键，帮助你更高效地创作。

## 画布操作

| 快捷键 | 操作 |
| --- | --- |
| `Space`（按住） | 拖拽平移画布 |
| 鼠标滚轮 | 缩放画布 |
| `Ctrl` / `⌘` + 滚轮 | 缩放画布（触控板捏合手势） |
| 触控板双指滑动 | 平移画布 |
| 触控板双指捏合 | 缩放画布 |

## 编辑操作

| 快捷键 | 操作 |
| --- | --- |
| `Ctrl + Z` / `⌘ + Z` | 撤销 |
| `Ctrl + Y` / `⌘ + Y` | 重做 |
| `Esc` | 结束当前直线绘制 |

## 绘制中操作

| 快捷键 | 操作 |
| --- | --- |
| 鼠标右键（画笔/橡皮模式） | 打开画笔详细设置面板 |
| `Shift`（套索矩形/椭圆模式） | 锁定为正方形 / 正圆 |

## 触控设备（iPad / 平板）

| 手势 | 操作 |
| --- | --- |
| 单指绘制 | 使用当前工具绘制 |
| 双指捏合 | 缩放画布 |
| 双指平移 | 移动画布 |

> 双指手势期间绘制会被自动取消，避免误触。
