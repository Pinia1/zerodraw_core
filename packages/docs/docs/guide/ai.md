# AI 生图

zeroDraw 集成了 AI 图片生成功能，可以根据你的描述和绘制内容生成或修改图片。

## 使用方式

AI 功能位于画布右侧面板中，包含两个标签页：

### Modify（AI 生图）

在文本框中输入你的描述（Prompt），AI 会基于你当前画布上的内容生成修改后的图片。

![Modify（AI](/docs/images/generate.gif)

**操作步骤：**

1. 在画布上绘制草图或打开已有内容
2. 在右侧「Modify」面板中输入描述文字
3. 点击生成按钮
4. 等待 AI 生成结果
5. 生成的图片会显示在面板中，你可以选择使用

### History（历史记录）

查看之前所有 AI 生成的结果，方便你回顾和重新使用之前的生成图。
