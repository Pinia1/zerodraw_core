# 绘图工具

顶部工具栏提供了所有绘图工具，从左到右依次为：

## 添加图片

点击工具栏最左侧的 **「+」** 按钮，可以从本地上传一张图片。图片会被创建为一个新的图层，方便你在上面继续绘制或作为参考。

## 画笔工具 🖊️

画笔工具是你最常用的绘制工具，包含两个子工具：

### 画笔（Pen）

自由绘制线条。点击画笔图标或展开面板后选择画笔。

**可调参数：**

| 参数 | 范围 | 说明 |
| --- | --- | --- |
| Size（尺寸） | 1 - 100 px | 笔触粗细 |
| Opacity（不透明度） | 0 - 100% | 笔触透明程度 |

**高级设置（右键画布或点击齿轮图标）：**

- **Hardness（硬度）**：控制笔触边缘的锐利程度，0 为最柔和，1 为最锐利
- **Stabilizer（平滑度）**：0 - 4，数值越大线条越平滑，适合画长曲线
- **Pressure（压感）**：开启后根据绘制速度模拟压感效果，线条粗细会随速度变化
- **Fill（填充）**：开启后笔触会被填充为实心

### 填充工具（Fill）

对封闭区域进行渐变填充。点击画笔展开面板后选择油漆桶图标。

**填充设置：**

- **Direction（方向）**：通过圆盘控件调整渐变角度（0° - 360°）
- **Color（色标）**：
  - 最多支持 4 个色标
  - 拖拽色标调整位置
  - 点击色标修改颜色
  - 支持添加和删除色标

## 橡皮擦工具 🧹

橡皮擦包含两个子工具：

### 像素橡皮擦（Pixel Erasure）

擦除画笔经过区域的像素内容，就像真实的橡皮擦。

**可调参数：**
- **Size**：1 - 100 px
- **Opacity**：0 - 100%

### 对象橡皮擦（Object Erasure）

用画笔划过图元素（线条、矩形、椭圆等），被画笔触碰到的整个图元素会被整体删除。适合快速清除不想要的笔划。

## 颜色选择 🎨

点击颜色按钮，可以选择当前绘图使用的颜色。画笔、图形、套索等工具都会使用此颜色。

## 图形工具 📐

图形工具包含三个子工具：

### 矩形（Rect）

拖拽绘制矩形。

### 椭圆（Ellipse）

拖拽绘制椭圆。

### 直线（Line）

点击设定起点，移动鼠标预览线段方向，再次点击设定终点。按 `Esc` 结束绘制。支持连续绘制：上一条线的终点会成为下一条线的起点。

**通用参数：**
- **Thickness（粗细）**：1 - 100 px
- **Opacity（不透明度）**：0 - 100%

## 套索工具 ✂️

套索工具用于选取画布上的区域，是进行区域操作的核心工具。

### 选区形状

| 形状 | 说明 |
| --- | --- |
| 自由选区 | 手绘围出任意形状的选区 |
| 矩形选区 | 拖拽框出矩形区域（按住 `Shift` 锁定为正方形） |
| 椭圆选区 | 拖拽框出椭圆区域（按住 `Shift` 锁定为正圆） |

### 选区操作模式

| 模式 | 说明 |
| --- | --- |
| 添加（Add） | 新选区与已有选区合并（默认模式） |
| 减去（Remove） | 从已有选区中减去新选区的重叠部分 |

### 选区动作

| 动作 | 说明 |
| --- | --- |
| 反选（Invert） | 反转当前选区，选中之前未选中的区域 |
| 复制到新图层（Copy） | 将选区内容复制到一个新图层 |
| 剪切到新图层（Clip） | 将选区内容剪切到新图层，原图层中对应区域被清除 |
| 清除选区（Clear） | 取消当前所有选区 |

## 位图化（Rope）

点击工具栏中的定位点图标，可以将当前图层的矢量内容**栅格化为位图**。这在你完成矢量绘制后想要锁定内容时很有用。
