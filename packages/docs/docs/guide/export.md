# 导出

zeroDraw 支持将你的作品导出为图片文件。

## 导出说明

导出时，zeroDraw 会：

1. 按图层顺序从底到顶合成所有**可见**图层
2. 正确应用每个图层的混合模式和滤镜效果
3. 输出高分辨率图片（默认宽度 1920px，等比缩放高度）

## 导出分辨率

| 计划 | 最大分辨率 |
| --- | --- |
| Free | 720p |
| Monthly / Quarterly | 标准 |
| Yearly | 4K |

## 注意事项

- 导出的图片包含画布背景色（如果开启了背景）
- 隐藏的图层不会出现在导出结果中
- 图层滤镜（模糊、亮度、对比度等）会被烘焙到导出图片中
- 混合模式会按照 Photoshop / CSS 标准正确渲染
