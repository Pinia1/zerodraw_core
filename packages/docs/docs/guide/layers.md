# 图层管理

zeroDraw 支持完整的多图层系统，让你可以分层管理绘画内容。

## 图层面板

图层面板位于画布左侧，包含两个标签页：

- **Layers**：图层列表
- **Assets**：素材资源（开发中）

## 基本操作

### 新建图层

点击图层面板右上角的 **「+」** 按钮，在弹出菜单中选择：

- **Empty layer**：创建一个空白图层
- **Reference picture**：上传一张参考图，会悬浮显示在画布上方

### 选择图层

点击图层列表中的某一项即可将其设为当前绘图图层。只有当前选中的图层才能进行绘制操作。

### 调整顺序

**桌面端**：直接拖拽图层项进行排序，上方的图层显示在最前面。

**移动端（iPad/手机）**：向右滑动图层项，露出拖拽手柄后拖拽排序。

### 显示 / 隐藏

每个图层都可以单独控制可见性。隐藏的图层不会参与渲染和导出。

> 当图层被隐藏时，画布上该图层内容会消失，且无法在该图层上绘制（光标变为禁止符号）。

## 图层属性

每个图层拥有以下可调属性：

### 不透明度（Opacity）

控制整个图层的透明程度，范围 0 - 100。

### 混合模式（Blend Mode）

控制当前图层与下方图层的颜色混合方式：

| 模式 | 英文名 | 效果 |
| --- | --- | --- |
| 正常 | Normal | 默认模式，直接覆盖 |
| 正片叠底 | Multiply | 颜色相乘，整体变暗，适合叠加阴影 |
| 滤色 | Screen | 颜色变亮，适合叠加高光 |
| 叠加 | Overlay | 亮部更亮，暗部更暗，增强对比 |
| 颜色减淡 | Color Dodge | 底色变亮，产生发光效果 |

### 滤镜（Filter）

zeroDraw 提供了多种图层级别的滤镜，可以对整个图层应用视觉效果：

**预设滤镜：**

| 预设 | 效果 |
| --- | --- |
| None | 无滤镜（默认） |
| Vivid | 鲜艳，增加饱和度和对比度 |
| Warm | 暖色调，偏暖黄 |
| Cool | 冷色调，偏冷蓝 |
| B&W | 黑白，完全去色 |

**自定义滤镜参数：**

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| Blur（模糊） | 0 px | 高斯模糊 |
| Brightness（亮度） | 100% | 大于 100 变亮，小于 100 变暗 |
| Contrast（对比度） | 100% | 调整明暗对比 |
| Saturate（饱和度） | 100% | 0 为完全去色 |
| Hue Rotate（色相旋转） | 0° | 旋转色相环 |
| Sepia（怀旧） | 0% | 模拟老照片效果 |
| Grayscale（灰度） | 0% | 转为灰度 |
| Invert（反色） | 0% | 颜色取反 |

## 背景控制

图层列表底部提供了画布背景颜色控制，你可以切换背景颜色或开关背景可见性。

## 参考图

通过图层面板的 **「+」→「Reference picture」** 添加参考图后，参考图会以浮窗形式显示在画布上方：

- **移动**：拖拽参考图窗口标题栏
- **缩放**：拖拽边缘/角落的控制手柄
- **关闭**：点击参考图窗口右上角的关闭按钮
