# 快速开始

欢迎使用 **zeroDraw**！这是一款运行在浏览器中的绘画工具，支持矢量绘制、位图编辑、多图层管理和 AI 辅助生图。

## 界面概览

打开 zeroDraw 后，你会看到以下主要区域：

| 区域 | 位置 | 说明 |
| --- | --- | --- |
| **工具栏** | 顶部居中 | 所有绘图工具、撤销/重做按钮 |
| **图层面板** | 左侧 | 图层列表，支持拖拽排序、新建、显示/隐藏 |
| **AI / 历史面板** | 右侧 | AI 生图（Modify）和历史记录（History） |
| **画布** | 中央 | 你的创作区域 |
| **缩放控制** | 右下角 | 缩放比例显示、放大/缩小、适应屏幕、全屏模式 |

## 基本操作

### 画布导航

- **缩放**：鼠标滚轮缩放画布；触控板双指捏合缩放
- **平移**：按住 `Space` 键拖拽画布；触控板双指滑动平移
- **适应屏幕**：点击右下角缩放控制栏中的「适应」按钮，画布将自动居中并缩放到合适大小
- **全屏模式**：点击右下角「全屏」按钮，隐藏左右面板获得更大画布空间

### 撤销与重做

- **撤销**：`Ctrl + Z`（Mac: `⌘ + Z`），或点击工具栏中的撤销按钮
- **重做**：`Ctrl + Y`（Mac: `⌘ + Y`），或点击工具栏中的重做按钮
- 支持最多 30 步历史记录

### 数据保存

你的绘制数据会**自动保存**到浏览器本地（IndexedDB），无需手动保存。切换页面或关闭浏览器后再回来，之前的内容会自动恢复。

> 不同项目的数据是隔离存储的，互不影响。
