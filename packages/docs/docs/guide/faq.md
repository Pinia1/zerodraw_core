# 常见问题

## 我的画会丢失吗？

不会。zeroDraw 会自动将你的绘画数据保存到浏览器本地存储（IndexedDB），每次操作后约 800ms 自动保存。即使关闭页面或刷新，下次打开时会自动恢复。

> 如果你清除了浏览器数据或使用了隐私模式，本地保存的数据会被清除。订阅用户可以使用云端同步功能。

## 支持哪些浏览器？

zeroDraw 基于 HTML5 Canvas（Konva.js），推荐使用以下浏览器：

- Chrome 90+（推荐）
- Edge 90+
- Safari 15+
- Firefox 90+

## 支持触控设备吗？

支持！zeroDraw 对 iPad 和平板设备做了专门优化：

- 单指绘制
- 双指缩放和平移
- 绘制中自动检测多指手势，防止误触
- 图层列表支持滑动操作

## 为什么画笔画出来不流畅？

试试调整以下参数：

1. 提高 **Stabilizer（平滑度）** 值（推荐 2-3）
2. 开启 **Pressure（压感模拟）** 获得更自然的线条
3. 调整 **Hardness（硬度）** 让笔触起止点更自然

## 图层太多怎么管理？

- 通过拖拽调整图层顺序
- 暂时不用的图层可以隐藏
- 使用套索工具的「复制到新图层」/「剪切到新图层」功能拆分内容
- 使用位图化功能（Rope）合并矢量内容为位图

## 什么是矢量和位图的区别？

在 zeroDraw 中：

- **矢量内容**：画笔线条、矩形、椭圆、直线等工具绘制的图形，放大不会模糊，可以被对象橡皮擦整体删除
- **位图内容**：上传的图片、AI 生成的图片、填充的色块，由像素组成

你可以使用工具栏的「位图化」按钮将矢量内容转换为位图。
